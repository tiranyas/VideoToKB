    - resolveLoomUrl: extracts CDN MP4 URL from page HTML, falls back to Apollo state

    - preprocessTranscript: collapses whitespace, formats timestamps as M:SS

     - Call `client.transcripts.paragraphs(transcript.id)` for paragraph-level text
     - Return TranscriptResult with text, paragraphs, wordCount
   - Export `preprocessTranscript(paragraphs: { text: string; start: number; end: number }[]): string`
     - Strip filler words: um, uh, like, you know, I mean, basically, actually, literally, right
     - Collapse multiple spaces to single space
     - Format timestamps as M:SS
     - Filter near-empty paragraphs (< 10 chars after cleaning)
     - Join with double newlines
   - Follow research Pattern 3 for preprocessing
